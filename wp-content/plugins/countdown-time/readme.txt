Display your event's date into a timer to your visitor with a countdown time block

== Description ==

Showcase your countdown timer in a professional way. This plugin adds a new block in the Block Editor by which you can create a professional-looking countdown timer block!

It is very effective in presenting your countdown timer nicely. There are many options that you can use to decorate your countdown timer in any color you like.


**[Demos](https://bblockswp.com/demo/countdown/)**


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.
- **Different Templates**: This block comes with different templates, you can easily change the templates.
- **Elements settings**: Change the elements settings such as show hide days, hours, minutes, seconds. Also, change the labels of them.


= How to use =
- First install the Countdown Time plugin
- Add the Countdown Time block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right side settings sidebar.
- Enjoy!

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Countdown Time**
3. Click on the **Countdown Time** block to add the block

= Download & Upload: =
1. Download the **Countdown Time** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`countdown-time.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Countdown Time** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
